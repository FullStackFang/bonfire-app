A person's live, self-reported state — the heart of Live Pulse. Two vocabularies share the component: ambient presence (`around` / `pool` / `asleep` / `out`) and spark participation (`in` / `otw` / `here` / `out`). Self-reported only — never GPS.

```jsx
<StatusPill status="here" where="at the bar" />
<StatusPill status="otw" time="8m" />
<StatusPill status="around" />
```

- Only `here` shows the breathing spark dot; `otw` shows an arrow. `out` is quiet grey — absence is stated softly, never as a flake record.
