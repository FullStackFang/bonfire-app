Bonfire's brand mark — a warm CSS teardrop flame. Use wherever the fire should appear: brand rows, primary CTAs, the "I'm in" affordance. Never decorative filler.

```jsx
<Ember size={18} glow />
```

- `glow` adds the ember bloom — reserve it for hero/brand moments, not inline runs.
- It's the only bespoke brand glyph; everything else uses Ionicons (see readme ICONOGRAPHY).
