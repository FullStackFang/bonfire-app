Pill-shaped label and filter chip. Selected state = `solid` (ember fill); unselected = `outline`. Never wrap numeric data in a chip — numbers get the display-italic typography treatment instead.

```jsx
<Chip variant="solid">now</Chip>
<Chip variant="outline">8:30pm</Chip>
<Chip variant="tinted" tint="var(--spark)">live</Chip>
```
